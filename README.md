# Emacs major mode for Slipshow

An emacs major mode for [slipshow](https://slipshow.org).

> [!WARNING]
> Do not use yet. This is early WIP. Require unreleased version of slipshow.

- Based on [`markdown-mode`](https://jblevins.org/projects/markdown-mode/)
- Uses [`eglot`](https://joaotavora.github.io/eglot/) for communication with Slipshow's LSP.
- Notable LSP addition:
  - Going forward and backward in the previewed presentation.
